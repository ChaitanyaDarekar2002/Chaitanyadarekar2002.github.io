## 📌 About This Project

This is my **personal portfolio website** built to showcase my skills, projects, dashboards, and experience as a **Data Analyst and Business Intelligence enthusiast**.

The website is designed to impress recruiters, clients, and hiring managers with a **modern dark UI**, smooth animations, and professional layout.

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 Dark Modern UI | Glassmorphism design with blue/purple color palette |
| ⚡ Smooth Animations | Scroll reveal, fade-up, and slide animations |
| ⌨️ Typing Animation | Auto-typing text in hero section |
| 📊 Skill Progress Bars | Animated bars triggered on scroll |
| 🔢 Counter Animation | Animated statistics numbers |
| 🌙 Dark/Light Mode | Toggle between dark and light themes |
| 📱 Fully Responsive | Works perfectly on mobile, tablet, desktop |
| 🤖 AI Chatbot | Portfolio assistant popup with quick replies |
| 📬 Contact Form | Working form via Formspree (sends to Gmail) |
| ⬆️ Back to Top | Floating button for easy navigation |
| 📄 Resume Preview | Embedded Google Drive PDF viewer |
| 🖼️ Dashboard Gallery | Real project screenshots with hover effects |
